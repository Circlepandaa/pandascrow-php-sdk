Official PHP SDK for [Pandascrow](https://pandascrow.com) Payment API - Simplified payments for Africa.

## 🚀 Features

- ✅ **Complete API Coverage** - Payments, Transfers, Verifications, Webhooks
- 🔐 **Secure** - API key & OAuth2 authentication, webhook signature validation
- 🎯 **Type-Safe** - PHP 8.1+ with strict typing
- 📦 **PSR-4 Compatible** - Follows PHP-FIG standards
- 🔄 **Retry Logic** - Automatic retries with exponential backoff
- 📊 **Logging** - PSR-3 compatible logging
- 🧪 **Tested** - Comprehensive test suite
- 📝 **Well Documented** - Full PHPDoc annotations

## 📋 Requirements

- PHP 8.1 or higher
- Composer
- Guzzle 7.0+

## 🛠️ Installation

```bash
composer require pandascrow/pandascrow-php-sdk
```
